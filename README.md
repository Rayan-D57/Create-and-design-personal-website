# Create-and-design-personal-website

Welcome to my personal portfolio! This project is designed to showcase my technical skills, academic background in computer engineering, and my journey through various engineering projects.

The website provides a professional and user-friendly interface that highlights my capabilities in Embedded Systems, Robotics, and Internet of Things (IoT).



## Tools and Technologies
The website was built using core web technologies to ensure fast loading, high performance, and cross-device compatibility:

*   HTML5: Used to build the website structure and organize content into clear, semantic sections.
*   CSS3: Used for interface design, styling, and modern visual effects (such as hovercard animations).
*   Responsive Design: Implemented using CSS Flexbox and Media Queries to ensure a seamless experience on both desktop and mobile devices.
*   Hosting: The website is hosted via InfinityFree.


## How I Built This Project

### 1. Planning
I structured the website logically to create an intuitive user experience:
*   Introduction: A brief overview of who I am.
*   Skills: Highlighting my technical expertise.
*   Projects: A showcase of my academic and personal engineering projects.

### 2. Design and Development
*   Modern UI: Utilized CSS Flexbox and Box Shadows to create clean, professional project cards.
*   Responsiveness: Used Media Queries to ensure the layout adapts perfectly to any screen size.

### 3. Deployment
*   Developed the code locally.
*   Created an account on InfinityFree.
*   Uploaded the project files via the File Manager in the hosting control panel to make the website publicly accessible.
